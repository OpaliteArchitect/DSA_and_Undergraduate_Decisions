# Undergraduate Decisions: C# Console Application Compilation

A collection of C# console applications demonstrating fundamental programming concepts, plus an interactive choice-based narrative game following the life of a college student.

## üìÅ Project Structure

### Data Structures and Algorithms
- **Activity1_1** ‚Äì Raw I/O without arrays or loops (20 individual variables)
- **Activity1_2** ‚Äì Same functionality using arrays and loops  
- **Activity1_3** ‚Äì Enhanced version displaying odd-numbered entries first
- **Activity2_1_2_2** ‚Äì `Student` struct implementation with fields (name, age, birthday, etc.)
- **Activity3_1** ‚Äì Basic `List<T>` proficiency
- **Activity3_2** ‚Äì List with fixed capacity (array-like behavior)
- **StackImplementation** ‚Äì Custom stack data structure
- **QueueImplementation** ‚Äì Queue implementation

### Classes and Inheritance
- **Classes** ‚Äì Inheritance hierarchy demonstrating OOP principles

### Creative Output: Undergraduate Decisions
An interactive narrative game following a college freshman's journey. Manage four resources: academic, mental, social, financial.
Follow one of two distinct narrative tracks, each leading to three dramatically different conclusions, offering six possible endings.

*This game is inspired by resource-management choice games like "Lapse: A Forgotten Future".*

## üöÄ Getting Started

### Prerequisites
- .NET 8.0 SDK or later
- Visual Studio 2022 or VS Code with C# extension

### Step 1: Clone or Download
```bash
git clone https://github.com/OpaliteArchitect/DSA_and_Undergraduate_Decisions.git
cd DSA_and_Undergraduate_Decisions

# Or download ZIP from GitHub
```

### Step 2: Build and Run
**Method 1: Visual Studio / VS Code**
- Open `Midterm_Compilation.sln`
- Press **F5** to run

**Method 2: .NET CLI**
```bash
# Build the solution
dotnet build

# Run specific project
dotnet run --project Midterm_Compilation
```
